# 项目说明

## 一个简单的塔防游戏，核心玩法就是放置防御塔，阻止怪物前进。
## 玩法特色：1. 包含肉鸽元素，每次放置塔是由击杀怪物金币自动触发的3选一，不能自主决定放什么塔，或者升级，后续会支持元素、天气、日夜、塔塔联动、颜色互动、战棋玩法等；
## 2. 塔也会有生命，怪物可以对塔造成伤害
## 3. 其他待定
